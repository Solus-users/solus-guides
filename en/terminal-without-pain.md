# Helpers with work
## Changing shell

```bash
sudo chsh -s /bin/zsh
```

# Install plugins

Add in your .zshrc rows:

```bash
source /usr/share/zsh-autosuggestions/zsh-autosuggestions.zsh
```
```bash
source /usr/share/zsh-syntax-highlighting/zsh-syntax-highlighting.zsh
```
